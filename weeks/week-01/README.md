## How to Use

1. **Open each notebook in Jupyter Lab or VS Code** (install Jupyter extension)
2. **Run every cell** — don't just read, execute and modify
3. **Break things on purpose** — change parameters, see what happens
4. **Complete all exercises** — solutions are provided but try first
5. **Keep the Syntax Reference open** in a second tab

## Setup

```bash
# Create a virtual environment
python -m venv week1-env
source week1-env/bin/activate  # or week1-env\Scripts\activate on Windows

# Install dependencies
pip install numpy pandas matplotlib seaborn scipy jupyter

# Launch Jupyter
jupyter lab
```
