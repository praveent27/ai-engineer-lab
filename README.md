# AI Engineer Lab 🧪

**Data Engineer → AI Engineer: A 28-Week Open-Source Learning Journey**

Live site: [https://praveent27.github.io/ai-engineer-lab/](https://praveent27.github.io/ai-engineer-lab/)

## What Is This?

A complete, documented roadmap from ML fundamentals to Microsoft AI-102 certification. Every week includes:
- Detailed study notes (GitHub Pages)
- Jupyter notebooks (run in Google Colab)
- Cheat sheets and syntax references
- Mini-projects and capstone projects
